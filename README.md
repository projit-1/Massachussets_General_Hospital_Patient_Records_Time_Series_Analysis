This project contains 25 beginner to intermediate time series analysis questions with complete Python solutions using a realistic healthcare dataset. It covers essential concepts such as data loading, datetime conversion, resampling, daily, weekly, monthly, quarterly, and yearly aggregations, trend analysis, moving averages, growth rates, visualization, encounter duration, claim cost analysis, categorical analysis, and executive KPI reporting. Each question includes practical code and business insights to help learners build a strong foundation in time series analytics.
