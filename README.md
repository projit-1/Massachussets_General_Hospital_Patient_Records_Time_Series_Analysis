Healthcare Time Series Analysis with Python
Project Overview

This project demonstrates an end-to-end Time Series Analysis workflow using a realistic healthcare dataset. The objective is to analyze patient encounter trends, identify seasonal patterns, detect anomalies, and forecast future healthcare demand using statistical and machine learning techniques.

The analysis is based on the Synthea Healthcare Dataset, primarily using the encounters.csv file, with additional information from patients.csv, organizations.csv, and payers.csv. Python was used for data cleaning, feature engineering, visualization, statistical analysis, and forecasting.

Project Highlights
Data loading and preprocessing
DateTime conversion and time indexing
Monthly, quarterly, and yearly trend analysis
Moving averages and rolling statistics
Time series decomposition (Trend, Seasonality, Residual)
Stationarity testing using the Augmented Dickey-Fuller (ADF) test
Lag feature engineering
Anomaly detection using Z-score and rolling statistics
Forecasting with ARIMA and SARIMA models
Machine learning forecasting using XGBoost
Model evaluation using MAE, RMSE, and MAPE
Professional visualizations and business insights

Key Outcomes

This project provides actionable insights into healthcare utilization patterns, seasonal fluctuations, and future patient demand. The forecasting models can support hospital administrators in capacity planning, resource allocation, staffing decisions, and operational planning. It also demonstrates practical skills in time series analysis, statistical modeling, forecasting, and data visualization, making it a strong portfolio project for Data Analyst, Business Analyst, and Junior Data Scientist roles.
